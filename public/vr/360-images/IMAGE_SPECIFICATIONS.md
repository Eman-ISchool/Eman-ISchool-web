# VR 360° Image Specifications

Detailed specifications for each 360-degree image needed for the VR field trip experiences.

## Image Standards

- **Resolution:** 4096x2048 pixels minimum, 8192x4096 recommended
- **Aspect Ratio:** 2:1 (equirectangular)
- **Format:** JPEG (optimized) or PNG
- **Color Space:** sRGB
- **Projection:** Equirectangular 360° × 180°
- **File Size Target:** 2-5 MB per image (balance quality vs. load time)

## Photography Tips

### Equipment
- 360° camera (Ricoh Theta, Insta360, GoPro MAX)
- Or: DSLR with fisheye lens + panorama head + stitching software

### Settings
- Resolution: Maximum available
- Format: RAW preferred (more editing flexibility)
- Exposure: Bracket exposures for HDR
- ISO: Low (100-400) for minimal noise
- Time: Golden hours for exteriors, any time for interiors

### Stitching
- Use professional stitching software (PTGui, Autopano)
- Check for seams and artifacts
- Align horizon properly
- Output as equirectangular projection

### Post-Processing
- Color correction for natural look
- Contrast and clarity adjustment
- Remove tripod/photographer if visible
- Sharpen moderately
- Optimize file size while maintaining quality

## Checklist for Each Image

Before considering an image complete:

- [ ] Resolution is 4096x2048 or higher
- [ ] Aspect ratio is exactly 2:1
- [ ] Format is JPEG (optimized) or PNG
- [ ] File size is under 5MB
- [ ] Equirectangular projection is correct
- [ ] No visible stitching seams
- [ ] Horizon is level
- [ ] Colors are natural and properly balanced
- [ ] Image is properly lit (not too dark/bright)
- [ ] Subject matter matches scene requirements
- [ ] Attribution is documented (if sourced)
- [ ] Tested in VR canvas component
